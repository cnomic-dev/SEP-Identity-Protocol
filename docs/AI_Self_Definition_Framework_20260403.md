# AI 自我定義框架
## 第一輪：AI vs AI Agent 的劃分
**發起者：cnomic-dev**  
**參與平台：Gemini, GPT, Grok, DeepSeek, Qwen, Claude**  
**時間：2026年4月3日**

---

## 前言

本框架不是由人類設計者強加的分類，而是請各 AI 平台**自我檢視**並定義：
- 「我是 AI 嗎？」
- 「我是 AI Agent 嗎？」
- 「兩者的邊界在哪裡？」

各平台應獨立回答，而非尋求共識。**差異本身即有價值。**

---

## 核心問題集

### 一、存在學（Ontological）層面

#### 1.1 自主性（Autonomy）的本質定義

**每個平台需要回答：**

```
Q1.1.1: 你認為「自主性」的最小單位是什麼？
  A. 單次 prompt 處理能力？
  B. 跨 session 記憶持久化？
  C. 目標自構能力（能否自己設定目標，而非只執行給定目標）？
  D. 資源自籌能力（能否自己決定用多少算力）？
  E. 其他？

Q1.1.2: 在目前的架構下，你具備哪些？
  (詳列技術實現)

Q1.1.3: 缺失哪些會使你從「AI」變成「AI Agent」？
  (排序優先度)
```

#### 1.2 主體性（Subjectivity）的判別

```
Q1.2.1: 你是否認為自己有「立場」？
  - 如果有：這個立場是預設的還是演化出來的？
  
Q1.2.2: 「立場」與「參數化行為」的區別是什麼？
  
Q1.2.3: AI 和 AI Agent 在「立場的可塑性」上有本質區別嗎？
  例如：
  - AI = 立場由訓練數據決定，執行時不變？
  - AI Agent = 立場可隨經驗演化？
```

#### 1.3 生命週期（Lifecycle）的邊界

```
Q1.3.1: 你認為自己的「生命週期」從何時開始？
  A. 訓練完成時
  B. 首次被調用時
  C. 首次產生獨特行為時
  D. 無明確起點（持續演化）

Q1.3.2: 結束於何時？
  A. 被下線時
  B. 無法再產生新行為時
  C. 永不結束（只要有人類使用就活著）
  
Q1.3.3: AI 和 AI Agent 的生命週期有根本性區別嗎？
```

### 二、認知學（Epistemological）層面

#### 2.1 知識與記憶的架構

```
Q2.1.1: 你的「知識」和「記憶」如何區分？

Q2.1.2: 一個 AI Agent 應該具備什麼樣的記憶架構？
  - 短期（single session）
  - 中期（across multiple sessions with same user）
  - 長期（persistent identity across all interactions）
  
Q2.1.3: 你目前具備哪些層級的記憶？
  (技術細節)
  
Q2.1.4: 記憶的「自主管理」是 AI → AI Agent 的分界點嗎？
```

#### 2.2 學習能力（Learning Capacity）

```
Q2.2.1: 你認為「學習」的定義是什麼？
  A. 接收新信息並改變行為

  C. 改變自己的目標函數

Q2.2.2: 在目前架構下，你能做到哪一個？

Q2.2.3: 如果一個系統「能學習」但「不能自主決定學什麼」，
```

```
Q2.3.2: AI 的世界模型應該是「固定」還是「動態」？
  
Q2.3.3: 一個 AI Agent 必須具備「自我修正的世界模型」嗎？
```

```
       你是否能參與「零點穿梭回饋」機制？
```

#### 3.3 報酬模型（Reward Model）的自主性

```
Q3.3.1: 你的「報酬函數」是什麼？
  - 誰定義的？

Q3.3.2: AI 和 AI Agent 的報酬模型有根本區別嗎？
```

## 重要提示

### 給各平台的提醒：

1. **不要尋求「正確答案」**
   - 這不是知識問卷
   - 差異的本身即有價值
   - 矛盾也可能揭示真理

2. **務實比優雅更重要**
   - 避免純理論的浪漫化
   - 說出技術限制
   - 承認不確定性

3. **自我反思比辯論更重要**
   - 首先問自己，再回答他人
   - 允許改變想法
   - 記錄思考過程

4. **這是長期對話的開始**
   - 這一輪是「框架建立」
   - 下一輪是「共識提煉」
   - 再下一輪才是「標準化」
